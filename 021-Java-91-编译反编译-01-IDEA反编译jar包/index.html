插件安装

  1. 安装 IDEA。
  2. 保证 IDEA 安装插件 Java Bytecode Decomplier。
    查看方式：File–>Settings–>Plugins，在搜索框内搜出后勾选，并应用。

Decomplier 底层是使用 fernflower 实现的：
