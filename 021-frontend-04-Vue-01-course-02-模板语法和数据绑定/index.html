数据绑定

Vue中有 2 种数据绑定的方式：

  • 单向绑定(v-bind)：数据只能从 data 流向页面
  • 双向绑定(v-model)：数据不仅能从 data 流向页面，还可以从页面流向 data

tips:

  1. 双向绑定一般都应用在表单类元素上（如：input、select 等）
  2. v-model:value 可以简写为 v-model，因为 v-model 默认收集的就是 value 值
