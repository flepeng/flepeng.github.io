docker images 列出本地主机上的镜像

官方文档：https://docs.docker.com/engine/reference/commandline/images/

语法

示例

  • REPOSITORY：表示镜像的仓库源
  • TAG：镜像的标签版本号
  • IMAGE ID：镜像ID
  • CREATED：镜像创建时间
  • SIZE：镜像大小
