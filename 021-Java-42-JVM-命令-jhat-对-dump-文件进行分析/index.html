命令 jhat 对 dump 文件进行分析

jhat 简介

jha命令与njmap 命令搭配使用，用于分析jmap生成的heap dump文件（堆转储快照）。jhat内置了一个微型的HTTP/HTML服务器，对生成的dump文件分析后，可以在浏览器中查看分析结果。

使用jhat命令，会启动一个http服务，默认端口7000。

注：jhat命令在 JDK9、JDK10 中已经被删除，官方建议用 VisualVM 代替。
