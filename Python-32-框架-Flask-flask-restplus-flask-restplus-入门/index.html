еҶҷflaskжҺҘеҸЈпјҢйңҖиҰҒеҗ‘еүҚз«ҜжҸҗдҫӣswaggerжҺҘеҸЈж–ҮжЎЈпјҢеӣ жӯӨйңҖиҰҒиҮӘеҠЁз”ҹжҲҗswaggerж–ҮжЎЈпјҢеҸҜд»ҘдҪҝз”Ё flask-restplus еә“гҖӮ

жҲ‘д»¬е№іеёёз”Ёзҡ„RESTfulйЈҺж јзҡ„ејҖеҸ‘дҫқиө–еҢ…жҳҜflask-restfulпјҢдҪҶжҳҜд»–дёҚиғҪиҮӘеҠЁз”ҹжҲҗ swagger ж–ҮжЎЈпјҢflask-restplusеҸҜд»ҘиҮӘеҠЁз”ҹжҲҗ swagger ж–ҮжЎЈпјҢдёӨиҖ…д№Ӣй—ҙе№¶жІЎжңүд»Җд№ҲеӨӘеӨ§зҡ„еҢәеҲ«пјҢflask-restfulжңүзҡ„дёңиҘҝе’ҢдҪҝз”Ёж–№жі•еңЁflask-restplusйҮҢйқўйғҪжңүгҖӮеҸҜиғҪдёәд»Җд№ҲеҸ«plusе°ұжҳҜиҝҷдёӘеҺҹеӣ еҗ§гҖӮ

жүҖд»Ҙд№ӢеүҚз”Ёзҡ„ flask-restful еҸҜд»ҘзӣҙжҺҘз”Ё flask-restplus зӣҙжҺҘжӣҝжҚўгҖӮ

0. е®ҳж–№ж–ҮжЎЈ Flask-RESTPlus 0.13.0 ж–ҮжЎЈ

  • е®үиЈ…(е·Іж”¶еҪ•)
  • еҝ«йҖҹејҖе§Ӣ(е·Іж”¶еҪ•)
  • е“Қеә”зј–з»„(е·Іж”¶еҪ•)
  • иҜ·жұӮи§Јжһҗ(е·Іж”¶еҪ•)
  • й”ҷиҜҜеӨ„зҗҶ
  • еӯ—ж®өжҺ©з Ғ
  • swaggerж–ҮжЎЈ(е·Іж”¶еҪ•)
  • йӮ®е·®
  • жү©еұ•жӮЁзҡ„йЎ№зӣ®
  • е®Ңж•ҙзӨәдҫӢ

1. е®үиЈ…flask-restplus

  1. йҮҮз”ЁеҰӮдёӢе‘Ҫд»Өе®үиЈ…flask-restplus

  2. е®үиЈ…е®ҢжҲҗеҗҺеҜје…ҘеҢ…ж—¶жҠҘй”ҷcannot import name 'cached_property'пјҢиҝҷжҳҜеӣ дёәж–°зүҲзҡ„werkzeugдёӯиҜҘжЁЎеқ—еҜје…ҘеҸ‘з”ҹдәҶеҸҳеҢ–пјҢйңҖиҰҒжүӢеҠЁдҝ®ж”№ /usr/local/lib/python3.7/site-packages/werkzeug/__init__.py ж–Үд»¶пјҢж·»еҠ пјҡ

2. дҪҝз”Ёж•ҷзЁӢ

Api()

  • apiпјҡдҪҝз”Ё Api() еҲӣе»ә api е®һдҫӢпјҢеҲӣе»әд№ӢеҗҺпјҢswagger ж–ҮжЎЈй»ҳи®ӨжҳҜејҖеҗҜзҡ„пјҢиҖҢдё”й»ҳи®ӨжҢӮиҪҪеҲ° / ж №и·Ҝз”ұ. и®ҝй—® swagger.json зҡ„ urlең°еқҖдёә /swagger.json
    • version: зүҲжң¬пјҢй»ҳи®ӨеҖј 1.0
    • titleпјҡж Үйўҳ
    • descriptionпјҡжҸҸиҝ°
    • prefix: Base URL: /v1
    • docпјҡй»ҳи®ӨеҸӮж•° doc='/', doc жҢҮе®ҡ swagger ж–ҮжЎЈең°еқҖжҢӮиҪҪзӮ№зҡ„, е°ұжҳҜйЎ№зӣ®еҗҜеҠЁеҗҺи®ҝй—®зҡ„и·Ҝеҫ„пјҢеҰӮпјҡhttp://localhost:8080/swagger/doc

жіЁж„Ҹ, еҰӮжһңжҳҜдҪҝз”Ё Flask е®һдҫӢ еҲқе§ӢеҢ–зҡ„ Api еҜ№иұЎ, йӮЈд№Ҳж–ҮжЎЈзҡ„ base_url дёә /, дҪҶжҳҜеҰӮжһңжҳҜдҪҝз”Ё и“қеӣҫ еҲқе§ӢеҢ–, е№¶дё”еңЁеҲқе§ӢеҢ–и“қеӣҫеҜ№иұЎж—¶и®ҫзҪ®дәҶ url_prefix еҸӮж•°, йӮЈд№Ҳж–ҮжЎЈзҡ„ base_url жҳҜжҢӮиҪҪеҲ°и“қеӣҫзҡ„ url_prefix дёҠзҡ„. д»ҘдёӢд»Јз ҒиЎЁзӨәе°Ҷж–ҮжЎЈжҢӮиҪҪеҲ°и“қеӣҫж №и·Ҝз”ұ, еҚі /api/v1.

namespace е‘ҪеҗҚз©әй—ҙ

  • api.namespace пјҡе‘ҪеҗҚз©әй—ҙпјҢеҫҲеӨҡжҺҘеҸЈйғҪжңүgetпјҢpostпјҢе‘ҪеҗҚз©әй—ҙжҠҠд»–д»¬еҲҶйҡ”ејҖпјҢеҸҜзҗҶи§Јдёәе°Ҹи“қеӣҫгҖӮ
    • pathпјҡжҠҠ path еҠ еҲ°и·Ҝз”ұең°еқҖзҡ„еүҚйқўпјҢй»ҳи®Өдёә namesapce зҡ„еҗҚз§°пјҢжңҖеҘҪеҶҷжҲҗ path='/'

namespace е’Ң и“қеӣҫ

еңЁ Flask дёӯжһ„е»әеӨ§еһӢ Web йЎ№зӣ®пјҢеҸҜд»ҘйҖҡиҝҮи“қеӣҫдёәи·Ҝз”ұеҲҶз»„пјҢе№¶еңЁи“қеӣҫдёӯж·»еҠ йҖҡз”Ёзҡ„и§„еҲҷпјҲurl еүҚзјҖгҖҒйқҷжҖҒж–Үд»¶и·Ҝеҫ„гҖҒжЁЎжқҝи·Ҝеҫ„зӯүпјүгҖӮ

иҖҢ Flask-RESTPlus зҡ„ class::Api е°ҶзӣҙжҺҘжҢӮеңЁеңЁи“қеӣҫдёӢйқўпјҢиҝҷд№ҲжҲ‘д»¬еҚіеҲ©з”ЁдәҶ Flask зҡ„и“қеӣҫиҝӣиЎҢеҜ№еҠҹиғҪжЁЎеқ—еҲҶзұ»пјҢд№ҹеҸҜд»ҘеҲ©з”Ё Api зҡ„зүҲжң¬еҜ№ Api зүҲжң¬иҝӣиЎҢз®ЎзҗҶпјҢеҜ№дәҺе°Ҹзҡ„жЁЎеқ—еҲҶзұ»пјҢжҲ‘д»¬еҸҜд»ҘеҲ©з”Ё Api зҡ„ namespace

и®ҝй—® swagger йЎөйқўж—¶пјҢurlең°еқҖдёә и“қеӣҫurl_prefix + /

з„¶еҗҺпјҢе°ұеҸҜд»ҘеҲӣе»әеҮәдёҚеҗҢзҡ„ namespaceпјҢжқҘзј–еҶҷиҮӘе·ұзҡ„ api д»Јз ҒдәҶгҖӮиҖҢеҸӘйңҖиҰҒеңЁ app е·ҘеҺӮдёӯжіЁеҶҢиҜҘ blueprintпјҢдҫҝеҸҜе°ҶиҮӘе·ұзҡ„зј–еҶҷзҡ„ api жҢӮиҪҪеҲ° flask app дёӯгҖӮ

и·Ҝз”ұ

ж·»еҠ и·Ҝз”ұжңүдёӨз§Қж–№ејҸ api.add_resource() ж–№жі•жҲ– api.route() иЈ…йҘ°еҷЁ

жӮЁиҝҳеҸҜд»Ҙе°ҶйғЁеҲҶи·Ҝеҫ„дҪңдёәеҸҳйҮҸеҢ№й…ҚеҲ°иө„жәҗж–№жі•гҖӮ

еҰӮжһңиҜ·жұӮдёҺеә”з”ЁзЁӢеәҸзҡ„д»»дҪ•з«ҜзӮ№йғҪдёҚеҢ№й…ҚпјҢFlask-RESTPlus е°Ҷиҝ”еӣһдёҖжқЎ 404 й”ҷиҜҜж¶ҲжҒҜпјҢе…¶дёӯеҢ…еҗ«дёҺиҜ·жұӮзҡ„з«ҜзӮ№еҜҶеҲҮеҢ№й…Қзҡ„е…¶д»–з«ҜзӮ№зҡ„е»әи®®гҖӮиҝҷеҸҜд»ҘйҖҡиҝҮеңЁжӮЁзҡ„еә”з”ЁзЁӢеәҸй…ҚзҪ®дёӯи®ҫзҪ®ERROR_404_HELPжқҘзҰҒз”ЁгҖӮFalse

еҸӮж•°и§Јжһҗ

parser дёҺ @api.expect(parser) иҒ”з”ЁпјҢдјҡеҜ№ header еӨҙдёӯеҜ№еә”еҸӮж•°иҝӣиЎҢж ЎйӘҢпјӣе№¶еңЁswagger зҡ„иҜ·жұӮеҢәеҹҹеҶ…ж·»еҠ еҜ№еә”зҡ„ header еҸӮж•°гҖӮ

дҪҝз”ЁRequestParserиҜҘзұ»иҝҳеҸҜд»Ҙе…Қиҙ№дёәжӮЁжҸҗдҫӣеҗҲзҗҶзҡ„й”ҷиҜҜж¶ҲжҒҜгҖӮеҰӮжһңеҸӮж•°жңӘиғҪйҖҡиҝҮйӘҢиҜҒпјҢFlask-RESTPlus е°Ҷе“Қеә” 400 Bad Request е’ҢзӘҒеҮәжҳҫзӨәй”ҷиҜҜзҡ„е“Қеә”гҖӮ

{'status': 400, 'message': 'foo cannot be converted to int'}

еҰӮжһңиҜ·жұӮеҢ…еҗ«и§ЈжһҗеҷЁжңӘе®ҡд№үзҡ„еҸӮж•°пјҢеҲҷи°ғз”Ё args = parser.parse_args(strict=True) еҸҜзЎ®дҝқеј•еҸ‘й”ҷиҜҜгҖӮ

ж•°жҚ®ж јејҸеҢ–

й»ҳи®Өжғ…еҶөдёӢпјҢиҝ”еӣһиҝӯд»Јдёӯзҡ„жүҖжңүеӯ—ж®өйғҪе°ҶжҢүеҺҹж ·е‘ҲзҺ°гҖӮиҷҪз„¶иҝҷеңЁеӨ„зҗҶ Python ж•°жҚ®з»“жһ„ж—¶ж•ҲжһңеҫҲеҘҪпјҢдҪҶеңЁеӨ„зҗҶеҜ№иұЎж—¶дјҡеҸҳеҫ—йқһеёёд»ӨдәәжІ®дё§гҖӮдёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢFlask-RESTPlus жҸҗдҫӣдәҶfieldsжЁЎеқ—е’Ң marshal_with()иЈ…йҘ°еҷЁгҖӮдёҺ Django ORM е’Ң WTForm зұ»дјјпјҢжӮЁдҪҝз”ЁиҜҘfieldsжЁЎеқ—жқҘжҸҸиҝ°е“Қеә”зҡ„з»“жһ„гҖӮ

дёҠйқўзҡ„зӨәдҫӢйҮҮз”ЁдёҖдёӘ python еҜ№иұЎе№¶еҮҶеӨҮе°Ҷе…¶еәҸеҲ—еҢ–гҖӮmarshal_with()иЈ…йҘ°еҷЁе°Ҷеә”з”Ёmodel. д»ҺеҜ№иұЎдёӯжҸҗеҸ–зҡ„е”ҜдёҖеӯ—ж®өжҳҜtask. иҜҘfields.Urlеӯ—ж®өжҳҜдёҖдёӘзү№ж®Ҡеӯ—ж®өпјҢе®ғйҮҮз”Ёз«ҜзӮ№еҗҚз§°е№¶еңЁе“Қеә”дёӯдёәиҜҘз«ҜзӮ№з”ҹжҲҗ URLгҖӮдҪҝз”Ёmarshal_with()иЈ…йҘ°еҷЁиҝҳеҸҜд»ҘеңЁ swagger и§„иҢғдёӯи®°еҪ•иҫ“еҮәгҖӮжӮЁйңҖиҰҒзҡ„и®ёеӨҡеӯ—ж®өзұ»еһӢе·Із»ҸеҢ…еҗ«еңЁеҶ…гҖӮгҖӮ

  • api.modelпјҡиҜ·жұӮеҶ…е®№пјҢжҲ–иҖ…жҳҜе“Қеә”еҶ…е®№пјҢйғҪеҸҜд»Ҙз”ЁmodelжҸҸиҝ°пјҢ

    • иҜ·жұӮеҶ…е®№дёҺ @api.doc(body=model) жҲ– @api.expect(model) иҒ”з”ЁпјҢ
    • е“Қеә”еҶ…е®№дёҺ @api.marshal\_with(token) жҲ– @api.response(200, 'Success', model) иҒ”з”Ё

  • model: е®ҡд№үеӯ—ж®өд»ҘеҸҠеӯ—ж®өзҡ„жҸҸиҝ°пјҢиҝҷдәӣеӯ—ж®өе№¶дёҚеҸӮдёҺеҸӮж•°жЈҖжҹҘпјҢиҖҢеҸӘжҳҜжёІжҹ“еҲ° api ж–ҮжЎЈдёҠпјҢжқҘж Үи®° api е°Ҷиҝ”еӣһд»Җд№Ҳз»“жһңпјҢд»ҘеҸҠеә”иҜҘжҖҺд№Ҳи°ғз”Ё apiгҖӮ

model еӯ—ж®өзұ»еһӢ

model жҸҗдҫӣдәҶеҫҲеӨҡзӣҙжҺҘеҜ№еә” Python ж•°жҚ®зұ»еһӢзҡ„еӯ—ж®ө, д»ҘдёӢдёәйғЁеҲҶеёёз”Ёеӯ—ж®ө:

  • fields.Integer иЎЁзӨәж•°еӯ—
  • fields.String иЎЁзӨәеӯ—з¬ҰдёІ
  • fields.Datetime иЎЁзӨәж—Ҙжңҹ
  • fields.List иЎЁзӨәеҲ—иЎЁ
    • fields.List(fields.String) е°ҶеҲ—иЎЁйЎ№еӯ—ж®өдј е…Ҙе…¶дёӯеҚіеҸҜ
  • etc

йҷӨдәҶиҝҷдәӣеёёи§„еӯ—ж®өпјҢеҰӮжһңдёҖдёӘеӯ—ж®өеҜ№еә”зҡ„еҖјжҳҜеҸҰдёҖдёӘжЁЎеһӢ, зұ»дјјдёҺж•°жҚ®еә“ дёҖеҜ№дёҖ жҲ–иҖ… дёҖеҜ№еӨҡ зҡ„е…ізі», еҸҜд»ҘдҪҝз”Ё fields.Nested еӯ—ж®ө. дҪҝз”Ёж–№ејҸеҰӮдёӢ:

еӯ—ж®өеұһжҖ§

зұ»дјјдәҺ fields.String жҲ–иҖ… fields.Integer зҡ„еӯ—ж®өйғҪжӢҘжңүиҮӘе·ұзҡ„еұһжҖ§, еҰӮжһңдҪҝз”ЁиҝҮ SQLalchemy зҡ„иҜқ, е®ғдёҺжЁЎеһӢзұ»зҡ„еӯ—ж®өеұһжҖ§жҳҜеҗҢдёҖдёӘжҖ§иҙЁ, еҸҜд»ҘеҜ№еұһжҖ§иөӢеҖј, жқҘжӣҙеҠ з»ҶиҮҙзҡ„жҸҸиҝ°еҪ“еүҚеұһжҖ§:

  • required з”ЁжқҘи®ҫзҪ®жҳҜеҗҰдёәеҝ…дј еҸӮж•°, й»ҳи®Өдёә False
  • readonly з”ЁжқҘи®ҫзҪ®еҸӘиҜ», иЎЁзӨәеҸӘжңүеңЁе“Қеә”зҡ„ж—¶еҖҷжүҚдјҡеҮәзҺ°, й»ҳи®Өдёә None
  • example з”ЁеҠӣи®ҫзҪ®зӨәдҫӢ, й»ҳи®Өдёә None
  • description еұһжҖ§з”ЁжқҘжҸҸиҝ°еҪ“еүҚеӯ—ж®өдҝЎжҒҜ, й»ҳи®Өдёә None

йғЁеҲҶеӯ—ж®өеұһжҖ§еҗҢж—¶д№ҹжҸҗдҫӣдәҶеҸӮж•°ж ЎйӘҢеҠҹиғҪ, еҜ№дәҺ fields.String:

  • enum дёҖдёӘеҲ—иЎЁ, еӯ—ж®өзҡ„еҖјеҸӘиғҪжҳҜе…¶дёӯзҡ„жҹҗдёҖдёӘ
  • min_length еӯ—ж®өжңҖе°Ҹй•ҝеәҰ
  • max_length еӯ—ж®өжңҖеӨ§й•ҝеәҰ
  • pattern дҪҝз”ЁжӯЈеҲҷеҲӨж–ӯ

еҜ№дәҺ fields.Integer fields.Float:

  • min жңҖе°ҸеҖј
  • max жңҖеӨ§еҖј
  • exclusiveMin: еҰӮжһңдёә True, еҲҷе·ҰејҖеҢәй—ҙ, й»ҳи®Өдёә False
  • exclusiveMax: еҰӮжһңдёә True, еҲҷеҸіејҖеҢәй—ҙ, й»ҳи®Өдёә False
  • multiple: еҖјеҝ…йЎ»жҳҜиҝҷдёӘж•°зҡ„еҖҚж•°

дҝқз•ҷйЎәеәҸ

й»ҳи®Өжғ…еҶөдёӢпјҢдёҚдјҡдҝқз•ҷеӯ—ж®өйЎәеәҸпјҢеӣ дёәиҝҷдјҡйҷҚдҪҺжҖ§иғҪгҖӮеҰӮжһңдҪ д»Қз„¶йңҖиҰҒеӯ—ж®өйЎәеәҸдҝқеӯҳпјҢдҪ еҸҜд»Ҙordered=True з»ҷдёҖдәӣзұ»жҲ–еҮҪж•°дј йҖ’дёҖдёӘеҸӮж•°жқҘејәеҲ¶йЎәеәҸдҝқеӯҳпјҡ

  • е…ЁеұҖ Apiпјҡapi = Api(ordered=True)
  • е…ЁеұҖ Namespaceпјҡns = Namespace(ordered=True)
  • жң¬ең° marshal()пјҡreturn marshal(data, fields, ordered=True)
