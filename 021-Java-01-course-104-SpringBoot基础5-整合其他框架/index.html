1гҖҒж•ҙеҗҲ SpringMVC

иҷҪз„¶й»ҳи®Өй…ҚзҪ®е·Із»ҸеҸҜд»ҘдҪҝз”ЁSpringMVCдәҶпјҢдёҚиҝҮжҲ‘д»¬жңүж—¶еҖҷйңҖиҰҒиҝӣиЎҢиҮӘе®ҡд№үй…ҚзҪ®гҖӮ

еҸҜд»ҘеңЁ application.yml ж–Үд»¶дёӯй…ҚзҪ®ж—Ҙеҝ—зә§еҲ«жҺ§еҲ¶пјҡ

1.1 дҝ®ж”№з«ҜеҸЈ

жҹҘзңӢSpringBootзҡ„е…ЁеұҖеұһжҖ§еҸҜзҹҘпјҢз«ҜеҸЈйҖҡиҝҮд»ҘдёӢж–№ејҸй…ҚзҪ®пјҡ

1.2 и®ҝй—®йқҷжҖҒиө„жәҗ

зҺ°еңЁпјҢжҲ‘д»¬зҡ„йЎ№зӣ®жҳҜдёҖдёӘjarе·ҘзЁӢпјҢйӮЈд№Ҳе°ұжІЎжңүwebappпјҢжҲ‘д»¬зҡ„йқҷжҖҒиө„жәҗиҜҘж”ҫе“ӘйҮҢе‘ўпјҹеӣһйЎҫжҲ‘д»¬еңЁдёҠйқўзңӢзҡ„жәҗз ҒпјҢжңүдёҖдёӘеҸ«еҒҡResourcePropertiesзҡ„зұ»пјҢйҮҢйқўе°ұе®ҡд№үдәҶйқҷжҖҒиө„жәҗзҡ„й»ҳи®ӨжҹҘжүҫи·Ҝеҫ„пјҡ

й»ҳи®Өзҡ„йқҷжҖҒиө„жәҗи·Ҝеҫ„дёәпјҡ

  • classpath:/META-INF/resources/
  • classpath:/resources/
  • classpath:/static/
  • classpath:/public

еҸӘиҰҒйқҷжҖҒиө„жәҗж”ҫеңЁиҝҷдәӣзӣ®еҪ•дёӯд»»дҪ•дёҖдёӘпјҢSpringMVCйғҪдјҡеё®жҲ‘д»¬еӨ„зҗҶгҖӮ

жҲ‘д»¬д№ жғҜдјҡжҠҠйқҷжҖҒиө„жәҗж”ҫеңЁclasspath:/static/зӣ®еҪ•дёӢгҖӮжҲ‘д»¬еҲӣе»әзӣ®еҪ• staticпјҢе№¶дё”д»Һ иө„ж–ҷж–Үд»¶еӨ№дёӯеӨҚеҲ¶ itcast.gif е’Ң test.js еҲ° static е°ұеҸҜд»Ҙи®ҝй—®дәҶгҖӮ

1.3 ж·»еҠ жӢҰжҲӘеҷЁ

жӢҰжҲӘеҷЁд№ҹжҳҜжҲ‘д»¬з»ҸеёёйңҖиҰҒдҪҝз”Ёзҡ„пјҢеңЁ SpringBoot дёӯиҜҘеҰӮдҪ•й…ҚзҪ®е‘ўпјҹжӢҰжҲӘеҷЁдёҚжҳҜдёҖдёӘжҷ®йҖҡеұһжҖ§пјҢиҖҢжҳҜдёҖдёӘзұ»пјҢжүҖд»Ҙе°ұиҰҒз”ЁеҲ° java й…ҚзҪ®ж–№ејҸдәҶгҖӮеңЁ SpringBoot е®ҳж–№ж–ҮжЎЈдёӯжңүиҝҷд№ҲдёҖж®өиҜҙжҳҺпјҡ

If you want to keep Spring Boot MVC features and you want to add additional MVC configuration (interceptors, formatters, view controllers, and other features), you can add your own @Configuration class of type WebMvcConfigurer but without @EnableWebMvc.
If you wish to provide custom instances of RequestMappingHandlerMapping , RequestMappingHandlerAdapter , or ExceptionHandlerExceptionResolver , you can declare a WebMvcRegistrationsAdapter instance to provide such components.
If you want to take complete control of Spring MVC, you can add your own @Configuration annotated with @EnableWebMvc .

зҝ»иҜ‘пјҡ

еҰӮжһңдҪ жғіиҰҒдҝқжҢҒSpring Boot зҡ„дёҖдәӣй»ҳи®ӨMVCзү№еҫҒпјҢеҗҢж—¶еҸҲжғіиҮӘе®ҡд№үдёҖдәӣMVCй…ҚзҪ®пјҲеҢ…жӢ¬пјҡжӢҰжҲӘеҷЁпјҢж јејҸеҢ–еҷЁ, и§ҶеӣҫжҺ§еҲ¶еҷЁгҖҒж¶ҲжҒҜиҪ¬жҚўеҷЁ зӯүзӯүпјүпјҢдҪ еә”иҜҘи®©дёҖдёӘзұ»е®һзҺ°WebMvcConfigurerпјҢе№¶дё”ж·»еҠ  @Configuration жіЁи§ЈпјҢдҪҶжҳҜеҚғдёҮдёҚиҰҒеҠ  @EnableWebMvc жіЁи§ЈгҖӮ
еҰӮжһңдҪ жғіиҰҒиҮӘе®ҡд№үHandlerMappingгҖҒHandlerAdapterгҖҒ ExceptionResolverзӯүз»„д»¶пјҢдҪ еҸҜд»ҘеҲӣе»әдёҖдёӘWebMvcRegistrationsAdapterе®һдҫӢ жқҘжҸҗдҫӣд»ҘдёҠз»„д»¶гҖӮ
еҰӮжһңдҪ жғіиҰҒе®Ңе…ЁиҮӘе®ҡд№үSpringMVCпјҢдёҚдҝқз•ҷSpringBootжҸҗдҫӣзҡ„дёҖеҲҮзү№еҫҒпјҢдҪ еҸҜд»ҘиҮӘе·ұе®ҡд№үзұ»е№¶дё”ж·»еҠ  @Configuration жіЁи§Је’Ң @EnableWebMvc жіЁи§Ј

жҖ»з»“пјҡйҖҡиҝҮе®һзҺ° WebMvcConfigurer е№¶ж·»еҠ  @Configuration жіЁи§ЈжқҘе®һзҺ°иҮӘе®ҡд№үйғЁеҲҶ SpringMvc й…ҚзҪ®гҖӮ

  1. еҲӣе»ә src\main\java\com\itheima\interceptor\MyInterceptor.java жӢҰжҲӘеҷЁпјҢеҶ…е®№еҰӮдёӢпјҡ

  2. е®ҡд№үй…ҚзҪ®зұ» src\main\java\com\itheima\config\MvcConfig.javaпјҢз”ЁдәҺжіЁеҶҢжӢҰжҲӘеҷЁпјҢеҶ…е®№еҰӮдёӢпјҡ

2 ж•ҙеҗҲjdbcе’ҢдәӢеҠЎ

spring дёӯзҡ„ jdbc иҝһжҺҘе’ҢдәӢеҠЎжҳҜй…ҚзҪ®дёӯзҡ„йҮҚиҰҒдёҖзҺҜпјҢеңЁ SpringBoot дёӯиҜҘеҰӮдҪ•еӨ„зҗҶе‘ўпјҹ

зӯ”жЎҲжҳҜдёҚйңҖиҰҒеӨ„зҗҶпјҢжҲ‘д»¬еҸӘиҰҒжүҫеҲ° SpringBoot жҸҗдҫӣзҡ„еҗҜеҠЁеҷЁеҚіеҸҜпјҢеңЁ pom.xml ж–Үд»¶дёӯж·»еҠ еҰӮдёӢдҫқиө–пјҡ

еҪ“з„¶пјҢдёҚиҰҒеҝҳдәҶж•°жҚ®еә“й©ұеҠЁпјҢSpringBoot е№¶дёҚзҹҘйҒ“жҲ‘д»¬з”Ёзҡ„д»Җд№Ҳж•°жҚ®еә“пјҢиҝҷйҮҢжҲ‘д»¬йҖүжӢ© MySQLпјӣеҗҢж ·зҡ„еңЁ pom.xml ж–Үд»¶дёӯж·»еҠ еҰӮдёӢдҫқиө–

иҮідәҺдәӢеҠЎпјҢSpringBoot дёӯйҖҡиҝҮжіЁи§ЈжқҘжҺ§еҲ¶гҖӮе°ұжҳҜжҲ‘д»¬зҶҹзҹҘзҡ„ @Transactional дҪҝз”Ёзҡ„ж—¶еҖҷи®ҫзҪ®еңЁеҜ№еә”зҡ„зұ»жҲ–ж–№жі•дёҠеҚіеҸҜгҖӮ
