0-PostgreSQL 安装（windows）

Windows 上安装 PostgreSQL

下载安装

官网下载地址：https://www.enterprisedb.com/downloads/postgres-postgresql-downloads

  1. 下载完成之后。双击下载安装包，开始安装



  2. 你可以修改安装路径

  3. 选择安装组件，不懂的选就是全部勾上

  4. 设置数据库路径

  5. 设置超级用户的密码

  6. 设置端口号，可以直接用默认就行

  7. 直接点 Next

  8. 去掉勾选，直接点 Finish

  9. 打开 pgAdmin 4

  10. pgAdmin 主页如下

  11. 点击左侧的 Servers > Postgre SQL 10

  12. 输入密码，点击 OK 即可

  13. 控制面板如下

  14. 打开 SQL Shell(psql)

远程访问

  1. 打开 postgresql 安装目录的data子目录

  2. 修改 pg\_hba.conf 文件：在IPV4部分添加新的一行:host all all 0.0.0.0/0 md5

  3. 控制面板–>系统与安全–>Windows防火墙,关闭防火墙。

4.重启服务
