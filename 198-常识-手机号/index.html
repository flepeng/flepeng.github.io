11 位手机号

手机号码，和自然人一样，实际上有着两套身份系统。一套叫MDN号，另一套叫IMSI号码 。

MDN（Mobile Directory Number，移动用户号码簿号码）

我们常说的手机号码，其实叫做MDN号码。主要是在国内使用。在中国大陆地区使用的手机号码组成是11位（其实是13位，但是通常情况下前两位不用输，默认为86）。

MDN号码的结构如下：CC国家码 + MAC移动接入码 + HLR识别码 + SN移动用户号 其中：

  • CC(country code)：国家码，2位。中国使用86。
  • MAC(mobile access code)：移动接入码，3位。这就是移动通信网网号。
  • HLR(home location register)：识别码，4位。实际上就是我们所说的归属地标识，最常见的也就是区号，比如青岛的0532，济南的0531，烟台的0535等等。由运营商统一分配。
  • SN(subscriber number)：移动用户号，4位。运营商内部用来区分用户的，也是我们的最终编号。

中国通信运营商有:移动、联通、电信，他们拥有的移动接入码如下所示：

  • 中国移动:134(0-8)、135、136、137、138、139、1440(物联网)、147(上网卡)、148(物联网)、150、151、152、157、158、159、165(虚拟运营商)、170(3,5,6)(虚拟运营商)、172、178、182、183、184、187、188、195、197、198
  • 中国联通:130、131、132、1400(物联网)、145(上网卡)、146(物联网)、155、156、166、167(虚拟运营商)、170(4,7,8,9)(虚拟运营商)、171(虚拟运营商)、175、176、185、186、196
  • 中国电信:133、1349(卫星通信)、1410(物联网)、149(上网卡)、153、162(虚拟运营商)、170(0-2)(虚拟运营商)、173、174(00-05)(卫星通信)、177、180、181、189、190、191、193、199
  • 中国广电:192
  • 4G号段:176/175(联通)、173/177(电信)、172/178(移动)
  • 暂未分配号段:142、143、154、161、164、194

IMSI（International Mobile Subscriber Identity，国际移动用户识别码）

IMSI号是一种“永久用户标识”，每一个手机SIM卡，对应一个IMSI号码。IMSI号，实际上等同于你的护照，主要是在国际使用。

那为何存在呢？

  • 在庞大的全球移动通信系统中，如何精准识别你的手机号？这个时候就需要一个特定的识别号码，以正确的识别移动用户，IMSI由此诞生。
  • 从技术上讲，IMSI可以区别出每个用户来自的国家，并且在移动网络中的所有位置（包括漫游区）都是有效的。因此，IMSI可以彻底解决国际漫游问题。
  • IMSI和手机号码是一一对应的，但是通信是转换成IMSI后进行的，而且两者之间的转换只有运营商才可以操作。也就是说，给你一组IMSI数据，等同于给了你一组手机号码，但是想给他们打电话，对不起，不行。

IMSI号码一般情况下是15位。IMSI号码由三部分组成：MCC移动国家码 + MNC移动网络码 + MSIN移动用户识别码。

  • MCC(Mobile Country Code)：移动国家码。这是IMSI的开头，是3位数字，表地区。地区号码的分配由国际电信联盟统一划分，中国的移动国家码为 460
  • MNC(Mobile Network Code)：移动网络码。这段表运营商。MNC是2-3位数字，中国的MNC是2位数字。
    • 中国移动MNC有：00、02、07、09；
    • 中国联通的MNC有：01，06；
    • 中国电信MNC有：03，05；
    • 中国铁通的MNC有：20。现在应该也是中国移动的了吧。
  • MSIN(Mobile Subscriber Identity Number)：移动用户识别码。这段用于识别网络中的用户，由运营商分配。MSIN是10-11位数字，我们国家的MSIN是10位。

在换卡是，我们换一张SIM卡实际上就是换了一个IMSI号码，而我们的MDN号码却一直没有改变。携号转网时，我们改动的也是IMSI号码。当然携号转网会牵扯到一些技术性的问题，这个有时间再聊吧。
