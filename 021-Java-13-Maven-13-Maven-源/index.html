为什么换源

远程仓库在国外，因为网络不稳定以及防火墙安全的原因，使用国外的 maven 不方便，所以我们需要换源。

仓库源

Maven 中央仓库地址

  1. http://www.sonatype.org/nexus/
  2. http://mvnrepository.com/ （本人推荐仓库）
  3. http://repo1.maven.org/maven2
