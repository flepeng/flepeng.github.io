01-fiddler 问题处理 The system proxy was changeed.Click to reenable capturing

详情

fiddler 一直显示黄色的：

原因

一定是有其他的服务一直在修改了代理，导致的fiddler频繁中断提示。

首先要找到对应的服务，可以通过（ProcessMointor）定位发现，很有可能是银行的控件和VPN相关的服务（SangforSP，SanforVPN Security Protect Service）在自动修改导致的。

解决思路

  1. 检查 Internet 选项里面的连接设置。

  2. 下载 Process Monitor，设置如下过滤操作条件

    看是哪个程序一直在修改。

  3. 找到对应软件之后关闭或者卸载即可。

    • 如果发现是 svchost.exe 不停的修改代理设置，很有可能是装了 EasyConnect，到服务里面关掉 SangforSP 就ok了。

    • 如果是银行的控件直接卸载即可，因为现在电脑端来说，使用不是很频繁，后续在有需要的话安装也很快

VPN相关的服务的禁用和调整为手动的方法（SangforSP，SanforVPN Security Protect Service）

  1. 先打开任务管理器（鼠标右击导航栏即可）

  2. 切换到服务菜单模块，右侧列表中就可以搜索找到（SangforSP，SanforVPN Security Protect Service）

  3. 找到SangforSP服务，现在状态列鼠标右击，选择停止当前服务，然后在启动类型列，鼠标右击选择属性，将启动类型修改为手动后保存即可

  4. SanforVPN Security Protect Service服务也按上面同样的方式修改即可，后续fiddler在抓包使用过程中就不会再出现这个搞人心态的黄色提醒了
