window 下 Gradle 的安装

在安装 Gradle 之前，先确认已经安装了JDK，需要是 jdk 1.8 或以上版本。

  • Gradle官方网站下载想要的版本，windows 下一般下载 complete 的压缩包。包名称一般为 gradle-8.1.1-all

  • 下载完成之后解压到你想要的目录。

  • 最后设置一下环境变量，将 Gradle 安装配置到操作系统环境中，主要就是将 PATH 增加 Gradle 的路径。

  • 都搞定后，验证一下，打开 cmd 输入 gradle -v 如何得到下面信息就说明配置成功了。

配置gradle仓库地址

Gradle 和 Maven 都是当前热门的自动化构建工具。Gradle 和 Maven 类似，也需要配置仓库地址。

Gradle 不像 Maven 一样可以配置 Setting 文件来修改本地库的位置，但是可以通过设置环境变量 GRADLE_USER_HOME 的路径来改变 Gradle 的本地仓库的位置。

Gradle 如果不配置环境变量，则会在用户主目录下（Windows下C:\\Users\\xxx目录）创建**.gradle** 目录，并将下载的依赖文件保存在这个目录里。
