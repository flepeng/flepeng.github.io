02-Luyten 工具反编译 JAR 包

安装

Luyten 下载之后直接可以使用。

官网：https://github.com/deathmarine/Luyten

  1. 进入网站后，找到 Releases：

  2. 下载Luyten。如果下载的是.exe的，直接双击就行了；如果是jar包，要在jar在的目录进入cmd命令，然后输入java -jar 要运行的jar包名，就行可以了。前提是配置了java开发环境。

反编译jar包

  1. 选择要反编译的jar包：
    File → Open File… → 选择jar包，会出现jar包里面所有的文件，每个文件对应一个节点，打开.class就能看到代码了。

  2. 生成反编译jar包的源码(.java文件)：
    File → Save As… → 选择保存的位置，这个是生成当前选中的文件的源码；
    File → Save All… → 选择保存的位置，这个是生成当前jar包所有文件的源码；
    cmd控制台会打印执行信息，执行信息停止就是完成了，生成后会有一个.zip的文件，解压后打开 **BOOT-INF → classes**，就能看到源码了。

设置正常显示中文

  • 勾选 Settings --> Enable unicode Replacement

设置显示全部引入包的路径,默认为*显示

  • 勾选 Settings --> Force Explicit Imports

使用感受

  1. 会和源码有较大区别。
    1. 很多对象都变为 final 修饰。
    2. 格式会变得臃肿。
