1、国内计算机类三大中文学报

国内计算机类三大中文学报：《计算机学报》、《软件学报》、《计算机研究与发展》。

共同点：

  • 都是EI核心来源期刊；
  • 中国计算机学会参与主办会刊；科学出版社出版发行；
  • 可接受8000-10000字左右的长文；
  • 稿量大，处理流程大多缓慢，应早投；
  • 国内众星捧月；
  • 基金论文比都很高；

1.1、《计算机学报》

  • 官网：http://cjc.ict.ac.cn/
  • 知网：https://navi.cnki.net/knavi/journals/JSJX/detail
  • 月刊，中国计算机学会与中国科学院计算技术研究所主办。
  • 投稿方式：网页登记然后Email投稿,中英文均可；初审后通知编号，邮寄打印稿二份及投稿声明等，
  • 审理费：150元。
  • 审稿周期：6个月左右。
  • 录用率：不详。
  • 版面费：不详。
  • 发表周期：不详。
  • 网上观点
    • 被拒的或录用的文章给出的意见都比较中肯，感觉审稿人专业啊。（感觉被拒的稿件处理快，可接受的稿件审理较慢）
    • 稿件须符合征稿指南及模板要求，不符合要求及逻辑不清、语句不通、影响阅读理解的稿件将退稿。请将稿件彻底准备好 ，没问题后再注册稿件编号，选择正确的投稿方向，注册后请在1天内提交论文，稿件一旦进入下一个处理环节，将不能更新。反复注册投稿无效。如有变化请及时告知。
  • 官网资料
    • 刊登的内容覆盖计算机领域的各个学科，以论文、技术报告、短文、研究简报、综论等形式报道以下方面的科研成果：计算机科学理论、计算机硬件体系结构、计算机软件、人工智能、数据库、计算机网络与多媒体、计算机辅助设计与图形学以及新技术应用等。
  • 知网资料
    • 基金论文比基本在0.98以上。

1.2、《软件学报》

  • 网址：http://jos.org.cn
  • 知网：https://navi.cnki.net/knavi/journals/RJXB/detail
  • 月刊，由中国科学院软件研究所和中国计算机学会联合主办，网站功能齐全，投稿处理流程合理。
  • 投稿方式：直接网站在线投稿，中英文均可。初审后通知编号，邮寄审理费。外审通过后再通知其它处理事宜。
  • 审理费：150元
  • 审稿周期：6个月左右。（有人前后总共投了二篇，第一篇4个月拒了，第2篇8个月给了通知，根据意见逐一修改，且必须给出修改说明，较为严谨）.
  • 录用率：不详
  • 发表周期：录用后的发表周期较长，需要耐心等待，但专刊较快。
  • 版面费标准：180.00元/面，收费比较厚道。
  • 网上观点
    • 整个处理流程中，编辑部会及时与作者沟通确认，感觉很受尊重。2007年（以后也会有）该刊组织了很多专刊，投专刊的文章被录用后发表周期相对短，但录用率超低（如今年网络专刊为5%）。专刊反映信息较快，是一种不错的方式。
    • 《模式识别与人工智能》比前者好中一些，但难度也不小。这个杂志是要求你寄两份打印稿，还有一份你的联系信息，然后大概一个星期左右他给你发email告知你通过初审，要求给审稿费100元，然后就是等待，大概三个多月给结果吧。每期文章大约20几篇，平均来说不是很好中，审稿一般4个月左右，发表周期就长了
  • 官网资料
    • 注重刊登反映计算机科学和计算机软件新理论、新方法和新技术以及学科发展趋势的文章，主要涉及理论计算机科学、算法设计与分析、系统软件与软件工程、模式识别与人工智能、数据库技术、计算机网络、信息安全、计算机图形学与计算机辅助设计、多媒体技术及其他相关的内容。
    • 基金论文比基本在0.90以上。
  • 知网资料
    • 基金论文比基本在0.90以上。

1.3、《计算机研究与发展》

  • 网址：http://crad.ict.ac.cn/
  • 知网：https://navi.cnki.net/knavi/journals/JFYZ/detail
  • 月刊，由中国科学院计算技术研究所-中国计算机学会联合主办,网站功能较为齐全。
  • 投稿方式：在线投稿或email投稿的同时还要邮寄3份打印稿，投稿声明等。好像不接受英文稿。
  • 审理费:200元。
  • 审理周期：官方说6个月左右（实际体会，投了一篇稿子，催了三次等了8个月拒了，郁闷。）
  • 录用率：不详。
  • 发表周期：不详。
  • 版面费：260元/页（不超过7页），超出的部分，每页400元。
  • 官网资料
    • 刊载内容：计算机科学技术领域高水平的学术论文，最新科研成果和重大应用成果。刊载内容：评估、计算机基础理论、软件技术、信息安全、计算机网络、图形图像、体系结构、人工智能、计算机应用、数据库技术、存储技术和计算机相关领域。
    • 征稿范围：计算机体系结构；计算机网络与通信；网络与信息安全；人工智能；软件技术；图形图像、信息检索等应用技术；及其他计算机相关领域。
  • 知网资料
    • 基金论文比基本在0.85以上。

1.4、《计算机科学技术学报》-英文名 JCST

必须要提的是：国内最权威的计算机科学与技术类期刊当属英文期刊JCST（中文名：计算机科学技术学报，双月刊，国内外同行评审，中国计算机学会和中科院计算所主办），SCI和EI双收录。

附JCST简历：《Journal of Computer Science and Technology》(JCST)是中国计算机科学技术领域唯一个英文学术性期刊。JCST为中国计算机学会会刊, 由中国科学院计算技术研究所承办。JCST由数十位国际计算机界的著名专家和学者联袂编审，把握世界计算机科学技术最新发展趋势。目前，JCST正在稳步地发展，其影响不断扩大，知名度日益提高。JCST荟萃了国内外计算机科学技术领域中有指导性和开拓性的学术论著，其中 部分文章邀请了著名计算机领域的专家撰写。其内容包括: 计算机科学理论，形式化方法，信息安全，算法与计算复杂性，计算机体系结构与高性能计算，模式识别与图像处理， VLSI设计与测试，软件工程，计算机网络与Internet，分布式计算与网格计算，自然语言处理，生物信息学，计算机图形学与人机交互，人工智能等。

  • 网站：http://jcst.ict.ac.cn/
  • JCST2005年SCI影响因子为0.353,在所有SCIE收录的5978种期刊中排名5136。

2、其他计算机方向的期刊

2.1、《计算机工程》（北核，CSCD） no

  • 主办单位：华东计算机技术研究所;上海计算机学会
  • 出版周期：月刊
  • 刊登内容： 热点与综述、人工智能与模式识别、先进计算与数据处理、网络空间安全、移动互联与通信技术、体系结构与软件技术、图形图像处理、开发研究与工程应用等。
  • 审稿周期：两个月
  • 发表周期：一年左右（可以加急发表）
  • 投稿费用：50元，好像70了，好像100了，加急审稿200
  • 投稿方式：E-mail，可在其网站上留言查询审稿情况，网站投稿http://www.ecice06.com/
  • 期刊专题：信息科技
  • 他人感受
    1. 有人说 “审稿周期大概三个月左右，主要收电子信息，科技，综合类文章。审稿速度在计算机类核心期刊中比较快。平均审稿时间两到三个月，审核难度较高，论文带有省级国家级项目编号更容易过”。
    2. 《计算机工程》中文核心，最大优点是审稿快。有基金号和项目编号的文章录用概率高。
    3. 论文质量还行，注重基金号
    4. 最大的优点就是审稿快，一般一个半月就能收到消息，两个月内肯定能收到消息。有基金号或者项目编号等的文章录用机率较高。
    5. 如不录用，审稿意见只有“不录用”三个字，无具体意见！
  • 知网资料
    • 基金论文比基本在0.98以上。

2.2、《计算机工程与应用》（北核，CSCD扩展）可

  • 网站：http://www.ceaj.org/
  • 知网：https://navi.cnki.net/knavi/journals/JSGG/detail?uniplatform=NZKPT
  • 主办单位：华北计算技术研究所
  • 出版周期：半月刊，每期130多篇文章
  • 审稿周期：宣称是三个月，其实编辑透露最长有四个半月的
  • 发表周期：9个月以上
  • 投稿费用：50元，
  • 版面费：1000/版，一般是两个版2000元，
  • 投稿方式：在线网站投稿
  • 期刊专题：电子信息科学综合
  • 他人感受
    1. 有人说 “极容易发表，基本上是只要是文字，都能发表，前提是交版面费”
    2. 也有人说 “初审到外审，整个审稿流程大概需要一到三个月，具体时间还是看审核人，审核难度一般，拒稿也不会给出详细的意见，感觉有点敷衍。机器语言学习，智能类的论文貌似更容易过一些。”
    3. 《计算机工程与应用》核心，比较好中，审稿速度比较慢
    4. 有网友反映以前被这个编辑部郁闷过一回，交了审稿费，如果被退稿，审稿意见都是千篇一律的一条：缺乏创新。这样其实不好，即便不录用，返回给作者一些有参考价值的审稿意见，应该是一个编辑部起码能够为作者做的一点事情。可在线投,审稿速度慢，一般3个月，快的有2个月的，慢的到了4个月。比较好中。审稿费￥50,版面费￥1000-1200,贵，有少于100元的稿费。
  • 知网资料
    • 基金论文比基本在0.94以上。

2.3、《计算机工程与设计》（北核） 可

  • 知网：https://navi.cnki.net/knavi/journals/SJSJ/detail?uniplatform=NZKPT
  • 主办单位：中国航天科工集团第二研究院706所
  • 出版周期：月刊
  • 办刊宗旨：本刊以传播新技术、促进学术交流为宗旨，坚持深度与广度、理论与应用相结合的方针，着力报道计算机前沿技术和热点技术，欢迎有创新和独立学术见解的学术论文，包括基金项目论文、获奖课题论文、学术会议优秀论文、博士和硕士论文等。
  • 审稿周期：三个月
  • 发表周期：一年左右(可以加急发表)
  • 投稿费用：100元，版面费比较高。
  • 投稿方式：在线网站http://www.china-ced.com/default.aspx 投稿，较为正规 。
  • 期刊专题：电子信息科学综合
  • 他人感受
    1. 初审很快，外审具体看审稿人，拖四五个月都是有可能的，优点是拒稿或者需要修改时会编辑指出问题所在，并给出意见。另外就是核心期刊的通病，排队等版面一般要半年到一年。”
    2. 《计算机工程与设计》和《计算机应用与软件》核心，相对来讲比较好中
    3. 审稿周期1个月，审稿费100元，比较好录用，论文质量一般。
    4. 反映比较亲睐有基金号或者项目编号等的文章。有网友反映给作者发来的修订意见竟然有两个错别字，汗。（曝光：感谢你对本刊借支的支持）看出哪个错别字啦吗？另外在投稿模版中，写着：“如果参考了本刊论文，请您在参考文献中列出，谢谢！”。
  • 知网资料
    • 基金论文比基本在0.93以上。

《计算机科学》曾用名《计算机应用与应用数学》（CSCD（拓展），北大核心） 重点

  • 水印和数据库相关的文章挺多的

  • 主办单位：重庆西南信息有限公司

  • 出版周期：月刊

  • 主要报道国内外计算机科学与技术的发展动态、涉及面广的方法论与技术、反映新苗头且能起承先启后作用的研究成果。

  • 审稿周期：两个月（快慢不等）

  • 发表周期：8个月到一年不等（有时候可以加急发表）

  • 投稿费用：无。

  • 投稿方式：邮寄打印稿，交付审稿费后可在其主页上查询稿件状态

  • 期刊专题：电子信息科学综合

  • 他人感受

    1. 审稿速度一般，出结果平均需要两个月，论文带有省级国家级项目编号的貌似审核会更快一点。出刊速度比较慢，平时快则半年，苦等了一年才有版面的也有。
    2. 审稿时间：2个月左右，审稿费300元，论文质量比较好，审稿非常严，录用难度：较难，容易被收到增刊去，注重基金号
    3. 比较注重较为新颖的文章，如果挂有基金号或者重大项目编号，一周内就能返回录用通知。在其网站上经常看到有当天收稿，第二天就录用的，估计是被杂志社约稿了，呵呵。这个期刊有一个好处就是不要审稿费了。文章质量还可以，部分学校曾经把它当权威？

  • 知网资料

    • 基金论文比基本在0.90以上。

  • 他人感受
    1. 审稿速度特别快，平均一个月左右就能收到审核结果。但是审稿难度较高，不过内容稍微不足可能会收录到增刊里面，增刊的录用率就比较高了，但是影响力要差很多，这点跟编辑交流的时候要问清楚。
      网络版会提前出刊，实体印刷版一般要等一年左右，有需求着急见刊的朋友投稿时要注意。
    2. 审稿时间：2个月左右，无审稿费，论文质量还行，录用难度：还行，容易被收到增刊去，注重基金号
    3. 用率始终保持在60%以上，不包括增刊，想上增刊，交钱（250元/页），1~2月就知道结果了！很喜欢把作者的稿件录用在增刊上，商业味道较浓，初步估计增刊录用率接近100%，正刊录用与否与文章质量关系不完全相关，有基金可能好点。不过该刊物许多高校不认可，不承认它是核心期刊，一般无课题或者非知名人士都是上增刊。
    4. 投稿时需要带上基金项目，至少是省级的科技类基金项目，国家自然科学基金更好，正常情况下，审稿周期是23个月，如果运气好，恰好遇到个审稿速度快的外审专家，1个多月也能返回结果（通常外审专家都是到最后期限才返回，摆架子，显得自己很牛掰），如果遇到多次返回修改，则需要36个月不等，顺利录用还好，如果不录用，只能白白耽误36个月的宝贵时间，版面费一般在25004000之间，杂志社会提供发票和纸质用稿通知书，录用后需要1年多见刊。

《计算机仿真》（北大核心）最水核刊

  • 投稿费用：审稿费100元，版面费250/页（上涨中…）
  • 杂志级别：中国核心期刊（统计源）
  • 稿酬回报：有，但不多
  • 他人感受：
    1. 电子投稿。审稿1～2个月，录用率为70％左右。较容易，基本上有仿真结果就能发表。

《数据采集与处理》（CSCD（拓展），北大核心） 不在 ccf 推荐里面

  • 网站：http://sjcj.nuaa.edu.cn
  • 杂志文章包含专业：主要涉及信号处理、通信、数据采集等内容
  • 投稿联系方式：南京御道街29号，210016
  • 投稿费用：每页160元，审稿费120元
  • 稿酬回报：200元左右
  • 他人感受：
    1. 投稿时可以在寄两份打印费时同时寄出审稿费120元，以加快审稿速度，审稿很认真，不管是否录用都回寄审稿意见，只要文章有些创新就可以录用，审稿周期在三个月左右。很多重点大学都很认可该刊物，在它还不是核心期刊时，就得到西安交大等学校的公认，是核心类期刊中比较正规、稿件质量较高的刊物之一。

《计算机应用与软件》（北大核心）

  • 主要栏目包括：最新技术动态、综合评述、软件技术与研究、数据工程、应用技术与研究、网络与通信、多媒体技术应用、人工智能与识别、图像处理与应用、嵌入式软件与应用、算法、安全技术、信息技术交流及其他相关内容。主要面向从事计算机应用和软件技术开发的科研人员、工程技术人员、高校师生等。

《计算机工程与科学》（CSCD（拓展），北大核心） 不在 ccf 推荐里面

  • 注重刊登计算机学科在理论、工程与应用等方面的研究论文、技术报告和科研成果，主要涉及计算机体系结构、并行处理、超级计算、人工智能、软件工程、计算机仿真、多媒体与可视化、数据库、计算机网络与分布式处理、计算机安全与保密、中文信息处理、微机开发与应用及其他相关内容。

  2. 计算机与数字工程（科技核心期刊，审稿时间 ：半个月，无审稿费，论文质量一般，录用难度：较易）。

推荐

将需要投稿的文章粗略地分为工程类文章（理论居多）和应用类文章（算法用于实际，比如基于CNN的猫狗识别）。

工程类文章需要投稿工程类期刊如《计算机工程》、《计算机工程与应用》、《计算机工程与设计》等。

应用类文章需要投稿《计算机应用》、《计算机应用研究》等期刊。

一篇论文发表与否有以下条件：基金，一作单位学历职称，最后才是文章质量（文章质量取决于文章创新度和写作手法，写作手法包括文章格式、语句是否书面化等），有部分计算机核心期刊对基金要求不严格，但是其余几条要求肯定更加严格。没有基金，你的论文要写的非常棒才行。建议让老师一作自己二作，论文反复修改精益求精。只是论文质量这一关就卡住很多人了，论文如果没有人引路全靠自己摸索，会走很多弯路。
